# Controle de Vendas em Python

Projeto desenvolvido durante meus estudos em Python, com foco em lógica
de programação, funções, validação de dados e organização de código.

## Objetivo
Simular um sistema simples de controle financeiro, permitindo:
- registro de vendas
- registro de baixas
- cálculo do total acumulado
- prevenção de saldo negativo
- verificação de meta financeira e sobra

## Estrutura do projeto
O projeto está organizado em etapas para demonstrar a evolução do aprendizado:

- Etapa 01: Meus primeiros passos na programação
- Etapa 02: Um programa simples baseado em logística
- Etapa 03: O início do meu sistema de vendas
- Etapa 04 em diante: Atualizações contínuas aderentes ao estudo.(comentários anexos ao código)

## Tecnologias utilizadas
- Python 3

## Status do projeto
Em desenvolvimento contínuo como parte do aprendizado em programação.
